How to run Project
1. Download and Unzip the file on your local system copy ecommerce .
2. Put ecommerce folder inside root directory (for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

Database Configuration

Open phpmyadmin
Create Database app_database
Import database app_database.sql (available SQL File Folder inside zip package)

Open Your browser put inside browser “http://localhost/ecommerce”
